#  Dugil Grill

Projeto de site estático para um restaurante de lanches e espetinhos, criado com HTML e CSS.

##  Sobre

Este repositório contém a estrutura de um site simples para **Dugil Grill**, um local fictício que apresenta cardápio de lanches, espetinhos e informações de contato — ideal para praticar desenvolvimento web básico com HTML e CSS. :contentReference[oaicite:1]{index=1}

##  Seções do Site

O site inclui:

- **Nossos Lanches** — descrições de opções de lanches.
- **Espetinhos na Brasa** — detalhes de espetinhos variados.
- **Contato** — telefone e endereço para pedidos.
